MapSVG WordPress map plugin: Vector maps, Google maps, Image maps. Turn any vector of jpeg/png image into an interactive map.

== Description ==

* [Demo](http://mapsvg.com/)
* [Tutorials](http://mapsvg.com/tutorials/)

= The last WordPress map plugin you'll ever need =
Create all 3 kinds of maps: Vector maps, Google maps, Image maps.

[youtube https://www.youtube.com/watch?v=E-si6gn9Ozk]

= What you can do with this mapping plugin =

* Interactive vector maps of countries (all countries are included in the plugin)
* Floorplans, maps of buildings, shopping malls
* Upload custom SVG images and turn them into a map
* Show you statistical data on the map with different shades of a color (Choropleth map)
* Google Maps
* Google Maps + clickable vector country overlay
* Google Maps + custom vector overlay
* Image maps from JPEG/PNG images (draw polygons on uploaded images right in the plugin)
* Add markers

= Add objects to your map =
Create objects with custom fields: users, cars, realty estates - and show them on a map as markers.

= Add markers to the map =
Add markers simply by clicking on a map

= Show tooltips, popovers, details view =
Show info in a tooltips, popovers or large details views.


== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/mapsvg-lite` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Go to Menu->MapSVG screen to create new map
4. Read online documentation at [mapsvg.com](http://mapsvg.com/documentation/) for more information
